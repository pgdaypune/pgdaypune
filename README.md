# PGPune

### Local dev setup

Install hugo https://gohugo.io/installation/

`brew install hugo`

**Run Locally**

`hugo server`

And visit http://localhost:1313/

### Add new meetups

```sh
hugo new content content/meetups/3.md
```

### Add new speakers

```sh
hugo new content content/speakers/full-name.md
```
